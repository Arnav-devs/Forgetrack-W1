# Forgetrack-W1
<h1>Personal Expense Tracker</h1>

# Features 
1. Add An Expense
2. View all Expenses
3. Filter By Category 
4. Exit

# add an expense
1. Date (dd//mm/yy)
2. Amount
3. Category (food/transport/entertainment/others)
4. Optional note

# view all expenses
